# Devlenko Portfolio

A responsive, single-page personal CV and portfolio website built with HTML5, Bootstrap 5, custom CSS, and vanilla JavaScript.

## Run locally

Open `index.html` in a modern web browser. No build step, package manager, or backend is required.

## Project structure

```text
index.html          Main semantic page structure and SEO metadata
css/style.css       Custom responsive visual design and animations
js/script.js        Navigation and scroll-reveal enhancements
assets/             Profile artwork and SVG favicon
```

## Design and accessibility

The page uses semantic landmarks, a keyboard skip link, descriptive image alt text, high-contrast calls-to-action, responsive Bootstrap navigation, and reduced-motion support. Metadata and clear section headings support search engines and answer-oriented discovery.
